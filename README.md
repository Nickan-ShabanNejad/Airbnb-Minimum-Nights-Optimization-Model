# Airbnb Minimum Nights Optimization Model

## Overview
This project analyzes Airbnb booking constraints and minimum-night stay requirements to improve rental efficiency and optimize occupancy strategies. The goal was to evaluate how minimum-night policies impact booking flexibility, utilization, and operational decision-making.

## Objective
The objective of this project was to design and analyze a structured data model capable of supporting optimization-based decisions for Airbnb rental management.

## Methods
- Designed entity relationship diagrams (ERDs) using Crow’s Foot notation
- Modeled relationships between listings, bookings, guests, and availability constraints
- Evaluated minimum-night stay policies and their impact on occupancy optimization
- Applied database design and operational analytics concepts to structure relational data efficiently

## Tools Used
- SQL
- Database Modeling
- Crow’s Foot ERD Design
- Excel
- Optimization Concepts

## Key Skills Demonstrated
- Relational database design

- Operational analytics
- Optimization thinking
- Business analytics

## Results
The project demonstrated how structured database systems and optimization strategies can support more efficient booking management and improve rental utilization decisions.
